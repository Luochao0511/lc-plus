## 介绍
公司的组件库，基础多方业务组件，常用api，常见问题栏目

## 特性
- 公共组件table表格，堪比excel一样，极致的虚拟列表
- 文件代码智能提示，ts编写方便使用者快速开发
- 组件库能够支持按需导入，完美配合TreeShaking
- 集成Vitepress，拥有优秀的文档案例，方便用户快熟开发

> ⚠ 建议在初次开发的时候详细阅读该文档

## 安装
```
pnpm install lc-plus
```
